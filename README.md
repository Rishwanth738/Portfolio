# Portfolio Website

This is my personal portfolio website designed to showcase my projects, skills, and professional background in a clean, responsive format.

## Live Deployment

**Website URL:** [https://rishwanth738-portfolio.netlify.app/](https://rishwanth738-portfolio.netlify.app/)

## Tech Stack

- **React** – Used as the primary framework for building the single-page application.  
- **Material UI (MUI)** – Implemented for responsive and accessible UI components with a modern aesthetic.  
- **EmailJS** – Integrated to enable contact form functionality and email delivery without requiring a custom backend.

## Overview

The portfolio includes:

- Project highlights with detailed descriptions  
- Skills and technology stack overview  
- Resume access and contact form  
- Smooth navigation and clean user experience
